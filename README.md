# User Enquiry System

A fullstack MERN application for managing user enquiries with CRUD operations.

## Features
- ✅ Create new enquiries
- ✅ View all enquiries in a table
- ✅ Update existing enquiries
- ✅ Delete enquiries
- ✅ Beautiful UI with Tailwind CSS

## Tech Stack
- **Frontend**: React + Vite + Tailwind CSS + Flowbite
- **Backend**: Node.js + Express
- **Database**: MongoDB

## Local Setup

### Backend
```bash
cd server
npm install
npm start
```

### Frontend
```bash
cd client
npm install
npm run dev
```

## Environment Variables
Create `.env` file in server directory:
```
DBURL=your_mongodb_connection_string
PORT=3000
```
